Move First

Move first. Scroll later.

Move First is a mobile app designed to improve digital wellbeing by requiring real-world physical activity before unlocking access to distracting apps like social media.

Instead of relying on timers or self-control alone, Move First creates a simple, powerful habit loop:

Effort → Reward → Repeat

By integrating with Strava, the app verifies completed workouts and conditionally unlocks selected apps only after a qualifying activity is completed.

✨ Features

🔗 Strava Integration
Securely connect your Strava account to track real activity

✅ Activity Verification
Unlock access only after completing a workout (e.g. 20+ minutes)

🔒 App Locking
Restrict access to selected apps until you’ve earned it

🔓 Conditional Unlocking
Apps become available after verified activity

🔁 Daily Streaks
Build consistency and reinforce positive habits

🎟️ Cheat System
Limited overrides to maintain flexibility without breaking the system

🎯 Purpose

Move First isn’t about blocking behaviour — it’s about reshaping it.

The goal is to help users become the kind of person who:

Moves before consuming

Earns their screen time

Builds healthier daily habits

🧠 Core Idea

Most digital wellbeing tools fail because they are easy to ignore or bypass.

Move First works differently:

Uses real, verified effort

Creates a clear reward loop

Encourages consistency through simplicity

🛠️ Tech Overview

Frontend: React Native / Flutter

Integration: Strava API (OAuth + activity data)

App Control:

iOS: Screen Time / Family Controls APIs

Android: Accessibility Services

Backend (optional): Firebase

🚀 MVP Scope

This project focuses on delivering a fast, functional MVP:

Connect to Strava

Detect qualifying workouts

Lock/unlock selected apps

Track daily streaks

No overengineering. No unnecessary features.

⚠️ Disclaimer

Due to platform limitations, app restriction behaviour may vary between iOS and Android. The MVP prioritises functionality and behavioural validation over full system-level control.

💡 Philosophy

This is not just a utility app — it’s a behavioural tool.

You don’t get to scroll by default.
You earn it.
